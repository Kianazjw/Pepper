# 花椒ER

* 整体项目基于ViewPager + RadioGroup + Fragment + DrawerLayout,MVC + Retrofit
* 使用Retrofit进行网络访问
* 运用 Animation 提升交互体验
* 运用LitePal操作数据库存储课表
* GitHub 项目地址 : https://github.com/Kianazjw/Pepper

花椒ER是华东交通大学Android客户端,包含课表、校内新闻、考试安排、成绩查询、电脑维修预约等功能模块。
